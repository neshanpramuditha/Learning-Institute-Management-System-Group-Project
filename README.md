# 📚 Learning Management System (LMS)

## 📖 Introduction

This project is a **Learning Management System (LMS)** designed for a learning institute. It provides a centralized platform where students can access academic services and institutional information in one place.

Students can:

* Register and log in
* View programs and course details
* Enroll in subjects
* Access personal academic information

Administrators can:

* Manage student records (update, filter, delete)
* Manage course and program information

The system also includes pages for:

* Program details
* Course information
* Vision & Mission
* Contact details

---

## 🎯 Goals and Objectives

* Digitalize manual processes of the institute

## ⚙️ System Architecture

The system follows a **Three-Layer Architecture**:

### 1. Presentation Layer

* HTML, CSS, JavaScript
* User Interface & Admin Panel

### 2. Business Logic Layer

* Authentication
* Enrollment processing

### 3. Data Layer

* MySQL Database
* Stores student, course, and enrollment data
